# Terragold's Content

A Mindustry mod that adds exciting new content to the game. The beginning is at its end.

## Installation
1. Open Mindustry
2. Go to Mods
3. Import from GitHub
4. Enter: Terragold-techie-underscore/terragolds-content
5. Enjoy!
