# Terragold's Content

A Mindustry mod that adds exciting new content to the game. The beginning is at its end.

## Installation
1. download the zip file
2. open mindustry!
3. open the mods thingy
4. hit "import"
5. hit "import from file"
6. click on the zip file
7. reboot mindustry
8. have fun
